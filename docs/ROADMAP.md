# 中学受験算数 網羅ロードマップ

「動くと理解が跳ねる」題材を単元横断で。✅=実装済 / 🔲=未実装。B=バックエンド(3D/2D)。
`type` は builder のキー。追加は `docs/ADD_SCENARIO.md` に従う。

## 単元: 立体図形(3D)※既存13問

切断(切り口の正六角形 ✅ / 積み木の切断 ✅)、展開(三角すい ✅ / 四角すい ✅ / 円すい ✅ / 円柱 ✅)、
ひもの最短 ✅、表面積(投影 ✅ / へこみ ✅ / 扇形の柱 ✅)、回転体 ✅、水位(おもり ✅ / 仕切り ✅)。

立方体の展開図 `cubeNet` ✅(十字・階段の2種。ヒンジ木で任意のnetに拡張可)、すいの体積×1/3 `pyramidVolume` ✅(立方体を3つの合同四角すいに分解)。

未実装(3D): 円すい台 `frustum`、容器を傾ける `waterTilt`、水位×時間グラフ `waterGraph`、
表面をはう最短 `netPath`、穴あき立体 `holeSolid`。展開図11種の残り(cubeNetのNETS表に追記するだけ)。
※詳細な立体図形リストは旧 `jukuse-3d` のROADMAP由来。優先: waterTilt → frustum。

## 実装の優先順位(提案)

「動きの効果が大きい × 頻出」順。**Claude Code はこの順で未実装(🔲)を埋めていくとよい。**

~~1〜8(saltBalance / turtleCrane / rolling / travelRound / treeDiagram / cubeNet / pyramidVolume / stream / passTrain)は実装済✅~~

次の候補(提案):
1. **時計算** `clock`(針の回転そのものが動き)
2. **割合の帯・面積図** `ratioBar`(割合単元の土台)
3. **等積変形** `equalArea`(頂点スライドで面積不変)
4. **容器を傾ける** `waterTilt`(3D水位の応用)
5. **周期・カレンダー** `cycle`(規則性の入口)

数値が実際の問題集に基づくものは、写真で正確な寸法をもらってから作ると精度が上がる。数値が「例」のものはタブ下 `source` に「例:」と明記する。
